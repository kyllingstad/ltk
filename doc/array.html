ltk.array

Array-related stuff.

T getElement (T, U)(T[U] array, U key, T dflt = T.init);
If the key exists in the associative array, its corresponding value is returned. If not, the given default is returned.

T[U] combine (T, U)(T[U][] arrays...);
Combine several associative arrays. The arrays are processed in the order they are given, and if a key exists in two or more arrays, the first one takes precedence.

T[U] dup (T, U)(T[U] a);
Duplicate an associative array.
